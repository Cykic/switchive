## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## ENV

Create .env file

```
# DB
DB_HOST= 'localhost'
DB_PORT= 5432
DB_USERNAME= 'user'
DB_NAME= 'db'
DB_PASSWORD= ''
DB_TYPE='postgres

#token
ACCESS_TOKEN_SECRET=""
JWT_EXPIRY=1d

# EMAIL
EMAIL=""
EMAIL_PASSWORD= ""


```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
